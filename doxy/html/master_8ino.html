Detailed Description

arduino master file

This is the file for master ESP8266 controller, which handles the request/data from

  • Wifi
  • push buttons (3)
  • Joystick
  • Accelerometer and perform output actions to:
  • OLED display
  • Vibration Motor
  • RGB LED
  • Buzzer
